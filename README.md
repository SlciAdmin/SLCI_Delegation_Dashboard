
# Delegation Dashboard (SLCI)

Voice-based task delegation system for companies.

Features:
- Admin and Employee authentication
- Admin can upload voice to assign tasks
- Voice converted to text (Whisper-ready placeholder)
- Tasks automatically assigned to employees
- Deadline reminders
- File/document uploads
- Employee marks task as done
- Admin dashboard to track completion

Tech Stack:
- Flask (Frontend + API)
- Python backend
- PostgreSQL database
- HTML/CSS/JS templates
- Whisper-ready speech-to-text module

Run:
1. Install requirements
2. Setup PostgreSQL
3. python app.py
